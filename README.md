# FizzBuzz実装

このリポジトリは、FizzBuzz問題を解く関数の実装を含んでいます。

## 機能

`fizzBuzz`関数は以下の仕様に従って実装されています：

- 1から指定された上限値までの数をプリント
- 3の倍数のときは「Fizz」
- 5の倍数のときは「Buzz」
- 3と5の両方の倍数のときは「FizzBuzz」
- それ以外の場合は数字をそのまま出力

## 使用方法

```javascript
// デフォルト値（100）で実行
const result1 = fizzBuzz();

// 任意の上限値（例：15）で実行
const result2 = fizzBuzz(15);
```

## テスト

テストを実行するには：

```bash
node fizzbuzz.js
```

以下のテストケースが含まれています：
1. limit=15での出力テスト
2. 通常の数字の出力テスト（limit=5）
3. デフォルト値（100）でのテスト
